Byiora — Digital Game Top-Up Platform for Nepal

Byiora is a secure and fast platform for delivering digital game credits and vouchers in Nepal. The system enables instant delivery without requiring user registration, supported by reliable payment integrations and a modern TypeScript-based architecture.

Features:

-Instant Delivery of game credits immediately after payment.
-Simple Checkout with no login required.
-Secure Processing supported by Supabase, RLS, and server-side validations.
-Optimized Frontend built with Next.js and TypeScript.

Tech Stack:

Frontend: Next.js, TypeScript
Backend: Supabase, PostgreSQL, PL/pgSQL
Deployment: Vercel
